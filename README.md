# Go Basics

Basic Syntax of Go Languange, divided into folders with proper documentation

&nbsp;

## Installation

```

git clone git@github.com:rttolipas/go_basics.git

```

```

cd go_basics

```

> Select a folder. Ex: 01_hello

```

cd 01_hello

```

```

go run main.go

```
